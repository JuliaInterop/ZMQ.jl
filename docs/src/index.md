# ZMQ.jl

*A Julia interface to ZeroMQ.*

**ZMQ.jl** is a [Julia](http://julialang.org) interface to [ZeroMQ, The Intelligent Transport Layer](http://zeromq.org).

## Package Features

- Access ZeroMQ sockets from Julia
